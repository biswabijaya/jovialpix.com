1. Create a database (for example, codexworld) at phpMyAdmin and import the "SQL/images.sql" file in the database.
2. Open the "DB.class.php" file on a editor ==> Specify the database host ($dbHost), username ($dbUsername), password ($dbPassword), and name ($dbName) as per your MySQL database credentials. 
3. Browse the "index.php" file on the browser and test the drag & drop reorder images functionality.
